# Requirements

## Software

- Python 3.9+
- Jupyter Notebook or Google Colab
- `7z` (p7zip) for extracting `.7z` archives

## Python Packages

- `numpy`
- `pandas`
- `scipy`
- `matplotlib`

## Notes

The notebook installs missing Python packages automatically at runtime. If `7z` is not installed, install `p7zip` and rerun the extraction cell.
